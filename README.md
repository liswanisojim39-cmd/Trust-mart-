# TrustMart - Zambia's Trusted Marketplace

TrustMart is a web-based application designed to connect Zambian buyers with local sellers, lodges, and service providers.

## Key Features
* **User Authentication:** Secure login via Firebase Auth.
* **Smart Posting:** Automated phone formatting (260+) for WhatsApp integration.
* **Directions:** One-click Google Maps navigation to business locations.
* **Admin Verification:** Manual review process for all listings to prevent scams.
* **Category Filtering:** Easy browsing for Shops, Garages, Hotels, and more.

## Technology Stack
* **Frontend:** HTML, CSS, JavaScript (Mobile-first design)
* **Backend:** Firebase Firestore (Database)
* **Storage:** Firebase Storage (Image hosting)
* **Deployment:** Firebase Hosting
